# 🎮 XO Arena — Документация & Руководство по запуску

Добро пожаловать в руководство по установке, настройке и запуску **XO Arena** — современной многопользовательской онлайн-игры в «Крестики-нолики» с полноценным чатом, системой профилей, ведением статистики и лобби для сражений в реальном времени.

Ниже приведены подробные инструкции по установке необходимых библиотек и запуску сервера для операционных систем **Windows** и **Linux**.

---

## 🛠️ Системные требования

Перед началом работы убедитесь, что на вашем компьютере установлен **Node.js** (рекомендуется версия **18.x LTS** или выше).
*   **Скачать Node.js:** [https://nodejs.org/](https://nodejs.org/)

---

## 📥 1. Сборка библиотек (Установка зависимостей)

Для работы приложения необходимы библиотеки, указанные в файле `package.json` (`express`, `ws`, `sqlite3`, `bcrypt`, `cors`). Чтобы собрать и установить их, выполните следующие шаги в терминале.

### Откройте терминал в папке проекта:
*   **Windows (PowerShell / CMD):** Откройте папку проекта в проводнике, зажмите клавишу `Shift`, кликните правой кнопкой мыши на пустом месте и выберите **«Открыть окно PowerShell здесь»** (или запустить Git Bash / Терминал Windows).
*   **Linux (Terminal):** Перейдите в каталог проекта с помощью команды `cd`:
    ```bash
    cd "/путь/к/папке/1_krestiki noliki"
    ```

### Выполните команду установки:
```bash
npm install
```
> **Совет:** Установка может занять около 1-2 минут, так как компилируются бинарные зависимости для шифрования паролей (`bcrypt`) и базы данных (`sqlite3`).

## 🚀 2. Инструкции по запуску

Вы можете запустить проект вручную из консоли или использовать готовые автоматические скрипты быстрого запуска, которые мы подготовили ниже.

---

### 💻 Вариант А: Инструкция для WINDOWS

#### Способ 1: Ручной запуск через PowerShell/CMD
В папке с проектом выполните команду:
```powershell
npm start
```
или напрямую:
```powershell
node server.js
```

#### Способ 2: Запуск в один клик (Базовый ярлык запуска)
Вы можете создать удобный файл автозапуска. Дважды кликнув по нему, вы автоматически установите недостающие библиотеки (если их нет) и запустите игру.
Создайте файл **`start_windows.bat`** в папке проекта с помощью блокнота и сохраните следующий код:

```bat
@echo off
title XO Arena Server
echo ==============================================
echo        XO Arena - Автоматический Запуск
echo ==============================================
echo.
cd /d "%~dp0"
if not exist node_modules (
    echo [INFO] Папка node_modules отсутствует. Устанавливаем библиотеки...
    call npm install
)
echo.
echo [OK] Запускаем сервер XO Arena...
echo Локальный адрес: http://localhost:3000
echo.
call npm start
pause
```

---

### 🐧 Вариант Б: Инструкция для LINUX

#### Способ 1: Ручной запуск через Терминал
Перейдите в директорию проекта и запустите Node.js сервер:
```bash
cd "/путь/к/папке/1_krestiki noliki"
npm start
```

#### Способ 2: Скрипт автоматического запуска (`.sh`)
Создайте файл **`start_linux.sh`** в корневой папке проекта со следующим содержимым:

```bash
#!/bin/bash
clear
echo "=============================================="
echo "       XO Arena - Автоматический Запуск"
echo "=============================================="
echo ""

# Переход в директорию скрипта
cd "$(dirname "$0")"

# Проверка библиотек
if [ ! -d "node_modules" ]; then
    echo "[INFO] Установка недостающих NPM библиотек..."
    npm install
fi

echo ""
echo "[OK] Запускаем сервер XO Arena на http://localhost:3000"
echo "Для завершения работы нажмите Ctrl + C"
echo ""

npm start
```

**Сделайте скрипт исполняемым:**
Перед первым запуском скрипта в Linux необходимо дать ему права на выполнение. Откройте терминал в папке проекта и введите:
```bash
chmod +x start_linux.sh
```
**Запуск скрипта:**
```bash
./start_linux.sh
```

## 🌐 3. Игра с другом на двух разных ПК (по сети)

Чтобы сыграть с другом, находясь за разными компьютерами, вы должны находиться в одной локальной сети (подключены к одному роутеру / Wi-Fi) либо настроить переадресацию портов (или использовать VPN-эмуляторы вроде Hamachi/Radmin VPN).

### Инструкция для владельца сервера (Хоста):
1.  Запустите сервер на своём компьютере любым удобным способом, описанным выше.
2.  Узнайте свой внутренний **IP-адрес в сети**:
    *   **В Windows:** Откройте командную строку или PowerShell, введите `ipconfig` и найдите строку **`IPv4-адрес`** (например, `192.168.1.15` или `192.168.0.104`).
    *   **В Linux:** Откройте терминал и введите команду `ip a` или `hostname -I`.
3.  Сообщите этот IP-адрес вашему другу вместе с портом `:3000` (например: `http://192.168.1.15:3000`).

### Инструкция для приглашенного игрока (Гостя):
1.  Откройте браузер на своём ПК.
2.  В адресную строку вставьте полученную от друга ссылку (например: `http://192.168.1.15:3000`).
3.  Зарегистрируйте аккаунт и перейдите во вкладку «В сети» справа.
4.  Вы должны увидеть друг друга в списке! Нажмите иконку **🎮** рядом с никнеймом друга, чтобы отправить ему запрос на дуэль.

## 📂 4. Структура проекта

*   `server.js` — Серверная часть приложения на Express и WS. Управляет WebSocket-подключениями, игровыми комнатами, чатом и сохраняет статистику в базу данных SQLite.
*   `database.db` — Локальная реляционная база данных SQLite (содержит информацию о профилях, хэши паролей и статистику матчей).
*   `index.html` — Структура интерфейса, модальные окна, игровая зона и блоки чата.
*   `app.js` — Клиентская логика: сетевые WebSocket-события, рендер интерфейса, обработка ходов на доске, логика лобби и уведомлений.
*   `style.css` — Современный UI на основе Glassmorphism (матовое стекло), переменные цветов, кастомные скроллбары и неоновая анимация победных комбинаций.
*   `.gitignore` — Защищает личную базу данных, системный мусор и папку `node_modules` от попадания в Git.
